# exposome-explorer-rs

Environmental exposure biomarker associations from Exposome Explorer.

## Source

- **URL:** `exposome-explorer.iarc.fr` (free download, no registration)
- **Files:** CSV/TSV exports of biomarkers, exposures, publications,
  biospecimens, measurement methods
- **License:** CC BY (IARC-hosted, open for research use)
- **Update cycle:** irregular (curated releases)

## Status

Planned — crate scaffold only. Implementation deferred until consumer need arises.
